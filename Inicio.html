Informacion general del OVI

Este OVI mostrará al estudiante el concepto de HTML5 y CSS, su conceptualización y la importancia que tiene este lenguaje de marca para el desarrollo de sitios web, así como los diferentes editores de código que hay en el mercado.

Objetivos del OVI

  • Comprender sistemas de control de versiones como GIT o GITHUB, para compartir y publicar archivos en formato HTML5 y CSS3, a través de linea de comando o por medio de la consola gráfica de GitHub Desktop. Ademas
  • Desarrollar sitios web en formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.

Tematicas

  • HTML5 y CSS
  • Estructura basica de html y Css
  • Aplicacion de elementos de html y css a un proyecto web
